# كيف ننشر التطبيق سوا على منصة مجانية (Render.com)

هذا التطبيق جاهز للنشر فوراً. اخترنا **Render.com** لأنها مجانية، بسيطة،
وما تحتاج بطاقة ائتمان للتجربة.

## الخطوات (نسويها مع بعض)

### 1) إنشاء حساب GitHub (لو ما عندك)
Render يحتاج الكود يكون على GitHub عشان ينشره تلقائياً.
- روح لـ https://github.com وسوّي حساب مجاني

### 2) ارفع هذا المجلد (webapp) لمستودع GitHub جديد
- بمتصفحك، بصفحة GitHub، اضغط "New repository"
- سمّه مثلاً: `sauce-monitoring-system`
- ارفع كل الملفات اللي بهذا المجلد (`app.py`, `requirements.txt`, `templates/index.html`)
  (تقدر تسحب وتفلت الملفات مباشرة بصفحة GitHub، ما يحتاج أوامر)

### 3) سوّي حساب على Render.com
- روح لـ https://render.com
- سجّل دخول بحساب GitHub نفسه (زر "Sign up with GitHub")

### 4) أنشئ "Web Service" جديد
- بلوحة Render، اضغط **New +** ثم **Web Service**
- اختر المستودع (Repository) اللي رفعته بالخطوة 2
- Render بيكتشف تلقائياً إنه تطبيق بايثون

### 5) إعدادات التشغيل (املأها بالضبط كذا)
| الحقل | القيمة |
|---|---|
| **Name** | sauce-monitoring (أو أي اسم تحبه) |
| **Runtime** | Python 3 |
| **Build Command** | `pip install -r requirements.txt` |
| **Start Command** | `gunicorn app:app` |
| **Instance Type** | **Free** |

### 6) اضغط "Create Web Service"
Render بيبدأ يبني وينشر التطبيق تلقائياً — ياخذ 2-5 دقايق أول مرة.

### 7) تم! بيديك رابط مباشر
بشكل: `https://sauce-monitoring.onrender.com`
افتحه بأي متصفح من أي مكان بالعالم — هذي لوحة تحكم حية فعلية.

## ملاحظات مهمة عن الخطة المجانية

- **التطبيق "ينام" بعد 15 دقيقة بدون استخدام** — أول طلب بعدها ياخذ 30-50 ثانية يصحى، وبعدها يشتغل عادي وسريع. هذا طبيعي بالخطط المجانية.
- **قاعدة البيانات (SQLite) تُمسح لو أعدت النشر (Redeploy)** — هذا مناسب تماماً لمرحلة التجربة الحالية، لكن قبل الاعتماد النهائي بنحتاج قاعدة بيانات دائمة (نفس تصميم PostgreSQL اللي عندنا بالمستند التقني).
- التطبيق الحالي **يستقبل بيانات عبر API فقط** — الكاميرا الحقيقية لسه ما مربوطة به مباشرة؛ هذا الربط خطوة قادمة بعد ما يثبت الكشف نفسه.

## بعد النشر — كيف نختبره سوا

بعد ما ياخذ رابطاً حياً، افتحه وجرّب:
1. تبويب "إضافة تحقق تجريبي" بأسفل الصفحة — عبّي اسم موظف وصوصات مطلوبة/مكتشفة واضغط إرسال
2. شاهد لوحة التحليلات تتحدث لحظياً فوق (دقة الموظف، التنبيهات)
3. أرسل لي رابط تطبيقك المنشور، وأقدر أساعدك تربطه لاحقاً بكود الكشف الحقيقي
